# Sync (Zed Theme)
A dark-minimalistic theme based on Radix Color System for Zed.

## Zed Settings
Removes a few icons that are not used for a minimalistic approach.
```
{
  "show_inline_completions": true,
  "base_keymap": "VSCode",
  "theme": "Sync Dark",
  "vim_mode": false,
  "ui_font_size": 14,
  "buffer_font_size": 14,
  "soft_wrap": "editor_width",
  "collaboration_panel": {
    "button": false
  },
  "chat_panel": {
    "button": false
  },
  "assistant": {
    "button": false
  },
  "notification_panel": {
    "button": false
  }
}
```
